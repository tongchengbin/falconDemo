
## FalconDemo
> 数据库开放使用 希望不要违规操作
#### 项目目录
```
─app
│  ├─views      视图函数

├─falconDemo    应用主目录
│
├─logs          日志文件
└─utils         工具类
    ├─cache     缓存操作封装
    │
    ├─conf      应用配置相关
    │
    ├─core      系统组件(包括中间件和异常等)

    │
    ├─db        数据库操作
```
